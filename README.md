# ExampleEshop

ExampleEshop is a **demo e-commerce website** built for portfolio purposes.  
It features a clean, responsive design with dynamic product listings and a shopping cart.

---

## Highlights

- **Dynamic Products:** Display products with images, prices, and categories.  
- **Search & Filter:** Quickly find items by name or category.  
- **Shopping Cart:** Add, remove, and update items with live total calculation.  
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop.  
- **Admin Panel (Demo):** Add products locally in your browser using localStorage.  

> ⚠️ The admin panel is a proof-of-concept. Products are stored in your browser only and are not visible to other users.

---

## Tech

- HTML, CSS, JavaScript  
- Flexbox & CSS Grid for layout  
- LocalStorage for cart and demo product management
